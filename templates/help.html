If you're new to ChronoWave and need help getting started, you've come to the right place. We've made it easy to help
yourself with our step-by-step guide which provides helpful guidelines and an easy walkthrough of how to upload you data,
analyze it, and visualize your results.

Step 1. Create a metadata.json file that describes your timeseries files.
Step 2. Compress your metadata.json and JSON, CSV, or Excel timeseries files in a single zip file.
Step 3. Navigate to the Upload Data tab.
Step 4. Click the "Choose File" button and select you zip file.
Step 5. Click Upload!

Example metadata.json. 
        {
            "name": "timeseries set name", // name of timeseries set 
            "timeseries": [
                {
                    "name": "timeseries one name", // name of timeseries 
                    "description": "timeseries description", 
                    "domains": ["domains"], 
                    "keywords": ["keywords"], 
                    "vector": {
                        "timestep_label": "datetime", // time column name in timeseries
                        "measure_labels": {
                            "measure": "unit" // non-time (column, unit) pairs in timeseries
                        }
                    },
                    "length": 5, 
                    "sampling_period": "1m", 
                    "training_filename": "training.csv", // filename containing training data 
                    "testing_filename": "testing.csv"  // filename containing testing data(optional)
                                                      // required for timeseries specified in forecasting task
                }
            ],
            "task": { // forecasting task description
                "forecast_period": "1m", 
                "count": 1, // number of forecast points to generate
                "timeseries_name": "timeseries_test" // name of timeseries to forecast for
            },
            "description": "description", 
            "domains": ["domain"], 
            "keywords": ["keyword"], 
            "contributors": ["contributor"], 
            "reference": "reference", 
            "link": "link" 
        }

You can download data from ChronoWave in just a few simple steps:
Step 1. Navigate to the Dowload Data tab.
Step 2. Select the time series data set you want to download from the drop down menu of available data sets.
Step 3. Choose the desired output format, it can be CSV, Exel, or JSON.
Step 4. Click the "Download" button to initiate the download of your time series data set in the chosen format.
Step 5. Wait for the download to complete. This may take a few seconds or minutes, depending on the size of your data set.
Step 6. Open the downloaded file in your preferred software, such as Microsoft Exel, Google Sheets, or a text editor.

To upload your forecast to ChronoWave, follow these steps:
Step 1. Navigate to the Upload Forecast tab on the Chronowave website.
Step 2. Fill out the form describing your forecast model name and contributors.
Step 3. Upload a single CSV, JSON, or Excel forecast file that has the same column names as the training dataset.
Step 4. Click Upload! Once your forecast is processed, it should be viewable on the Performance Metrics page.
